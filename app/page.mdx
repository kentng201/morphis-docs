# Morphis Documentation

Build TypeScript APIs with Laravel-style structure, Bun-first tooling, and a clean developer experience.

## What is included

- Opinionated project structure
- Routing with decorators and middleware
- Validation and transformers
- Model and database abstractions
- CLI-driven scaffolding and deployment helpers
